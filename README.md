# Task Summary
This repository contains my work for Task 1 of the Prompt Engineering Internship at Future Interns. The task requires creating an AI-assisted YouTube video using a selected niche.  
For this task, I chose the niche: Cybersecurity for CS students.

According to the Future Interns program guidelines, this task requires:  
- Prompt chain documentation  
- Full video script (800–1500 words)  
- Thumbnail text + AI-generated image or mockup  
- Final video generated using AI tools
